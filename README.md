# Error Analysis Framework for Object Detection Systems

## Overview
This project is a Python-based framework for analyzing the performance of object detection systems. It evaluates detection results by identifying different types of errors and provides insights to improve model accuracy.

## Features
- Error analysis for object detection models
- Detection performance evaluation
- Generation of error reports
- Visualization of analysis results

## Technologies Used
- Python
- PyCharm
- OpenCV
- NumPy
- Matplotlib

## Project Structure
- code/ - Source code
- results/ - Generated analysis reports
- main.py - Main program

## Dataset
The dataset used for this project is not included in this repository because of its large size.
